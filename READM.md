# AttachmentControl

A custom PowerApps Component Framework (PCF) control developed by **Shaheer Ahmad**.

## Description

The **AttachmentControl** is designed to enhance attachment handling capabilities within Power Apps. This component provides a seamless way to manage files with improved user experience and functionality compared to standard controls.

## Features

-   **File Management**: Efficiently handle file attachments.
-   **Customizable**: Built with extendability in mind.
-   **Seamless Integration**: Designed to work natively within Power Apps Canvas and Model-Driven apps.

## Installation

1.  **Download**: Get the latest solution file from the [Releases](https://github.com/shaheerahmadch/Canvasify-Components/releases) page (if available) or build from source.
2.  **Import**: Import the solution into your Power Apps environment.
3.  **Add to App**: Open your Power App, go to the **Insert** menu, find **Code Components**, and select **AttachmentControl**.

## Usage

1.  Select the **AttachmentControl** component on your canvas.
2.  Configure the properties in the property pane (e.g., `files`, `enableList`).
3.  Bind the necessary data fields to the component.

## Support & specific details

For more specific details on properties and configuration, please refer to the `ControlManifest.Input.xml` file.
